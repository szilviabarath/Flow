Flow

Quick to install thanks to the various options.

Easy to use with the help of the built-in classes.

CSS and JavaScript solutions for targeting common overflow issues.

Make elements fit perfectly by adding some simple CSS classes.

Get started

Install

via package manager

npm i @miniscott/flow

via CDN

CSS:

https://cdn.jsdelivr.net/gh/szilviabarath/flow-css@main/style.css

JS:

https://cdn.jsdelivr.net/gh/szilviabarath/flow-css@main/overflow.js

Download

Setup

  1. Choose from one of the install options.
  2. Check overflowing elements with the overflow-checker in your console.
  3. Choose the class from the classlist that is most likely to solve you overflow problem.
  4. If you are not sure which class to use, jump to the example uses.
  5. Add the class to your HTML element. As simple as that!

Examples

Fixed-width elements

Use the class overflow-all on your HTML element to fix overflow for fixed width elements.

HTML:

<div class=fixed-width-element></div>

add CSS class:

<div class=fixed-width-element overflow-all></div>

Flexbox without wrapping

Use the class wrap on your HTML element to fix overflow for fixed width elements.

HTML:

<div class=flex></div> <div></div>

add CSS class:

<div class=flex wrap></div> <div></div>

CSS grids

Use the class minmax on your HTML element to fix overflow for fixed width elements.

It is good practice to use fr units with your grids. Read more about it here.

HTML:

<div class=grid1></div>

add CSS class:

<div class=grid1 minmax overflow-all></div>

Negative margins

Use the m-0(set margin to zero) and overflow-all classes on your HTML element-

HTML:

<div class=element></div>

add CSS class:

<div class=element overflow-all></div><p class=m-0></p>

Images without max-width

Use the class minwidth on the parent element and the class responsive-container on the image to fix oveflow of images without max-width.

HTML:

<div class=img-container><img class=fixed-width-img>/div>

add CSS class:

<div class=img-container minwidth></div>img class=fixed-width-img responsive-container>/div>

Injected ads

Since this problem occurs when the container is wider than the parent element, it is advised to use the overflow: hidden and the wrap classes on the parent element and the responsive-container class on the overflowing element.

HTML:

<div class=parent-element></div><div class=responsive-container></div>

add CSS class:

<div class=parent-element overflow-hidden wrap></div><div class=responsive-container></div>
